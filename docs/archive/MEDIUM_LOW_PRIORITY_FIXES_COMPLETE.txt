## Usage Examples

### Add Custom Post Type
```python
frappe.get_doc({
    "doctype": "Post Type",
    "type_name": "Product Launch",
    "description": "Special posts for product launches with enhanced features",
    "icon": "fa-rocket",
    "color": "#FF6B6B"
}).insert()
```

### Add Custom Media Type
```python
frappe.get_doc({
    "doctype": "Media Type",
    "type_name": "3D Model",
    "file_extensions": "obj, fbx, gltf, glb",
    "max_file_size_mb": 100,
    "icon": "fa-cube",
    "color": "#4ECDC4"
}).insert()
```

### Add Custom Blast Type
```python
frappe.get_doc({
    "doctype": "Blast Type",
    "type_name": "Regional Rollout",
    "description": "Publish to regions in sequence",
    "requires_scheduling": 1,
    "supports_time_zones": 1,
    "icon": "fa-map",
    "color": "#95E1D3"
}).insert()
```

### Set Custom Sync Frequency
```python
# Sync analytics every 6 hours
settings = frappe.get_single("Marketing Hub Settings")
settings.sync_frequency = 6  # hours
settings.save()
```
