## 1. ✅ CRITICAL: Python channel_specs Dictionary

**Before**: Hardcoded dictionary in content_orchestration.py
```python
channel_specs = {
    "SMS": {"max_chars": 160, "no_html": True},
    "Meta Ads": {"max_chars": 125, "no_html": False},
    # ... 8 more hardcoded entries
}
```

**After**: Dynamic query from Social Media Network doctype
```python
network = frappe.get_cached_doc("Social Media Network", channel)
specs = {
    "max_chars": network.max_text_length or 5000,
    "no_html": not network.supports_html,
    "max_hashtags": network.max_hashtags or 30
}
```

**Impact**: 
- ✅ Can now update platform specs without code deployment
- ✅ Added: supports_html, max_hashtags, max_mentions, max_images, max_videos fields
- ✅ Updated all 15 existing networks with correct specs
- ✅ Platform limit changes no longer require code changes
