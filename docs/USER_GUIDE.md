# Marketing Hub - User Guide

## Getting Started
Marketing Hub helps you manage your marketing efforts directly within ERPNext.

### 1. Connect Platforms
Go to **Marketing Connect** workspace to set up your integrations:
- **Google Ads**: Enter Client ID and authorize.
- **Meta Ads**: Connect your Business Manager.
- **LinkedIn Ads**: Authenticate your Ad Account.

### 2. Define Segments
Use **Segments** to create dynamic audience lists:
- Filter Leads by Source, Status, or Interest.
- Filter Customers by Territory or Purchase History.
- Preview the audience size instantly.

### 3. Create Content
Upload images and write copy in the **Content** library. Approved content can be reused across multiple campaigns.

## Core Workflows

### Launching a Campaign
1. Navigate to **Campaigns** > **New Campaign**.
2. Set a Name, Budget, and Date Range.
3. The campaign will now track all linked activities.

### Sending an Omni Blast
1. Go to **Omni Blast** in the sidebar.
2. Select your Campaign and Audience Segment.
3. Choosing Channels: **Email**, **SMS**, or **WhatsApp**.
4. Compose your message (or select a template).
5. Schedule it or Send Immediately.

### Analyzing Results
- **Dashboard**: See high-level spend and ROI.
- **Campaign Performance Report**: Detailed breakdown of leads and cost per acquisition.
- **Channel Attribution**: See which platforms drive the most value.

## Settings
Configure global settings in **Marketing Hub Settings**:
- Enable/Disable specific channels (e.g., SMS, WhatsApp).
- Set default Gateway providers.
